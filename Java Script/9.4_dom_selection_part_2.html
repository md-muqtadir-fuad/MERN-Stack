Using querySelector and querySelectorAll

The querySelector method allows you to select the first element that matches a specified CSS selector. For example:

const firstParagraph = document.querySelector('p');

This will select the first <p> element in the document.

The querySelectorAll method returns a static NodeList of all elements that match the specified CSS selector. For example:

const allParagraphs = document.querySelectorAll('p');

This will select all <p> elements in the document.
